# Local AI Insight Pipeline

Prototype for local analysis of humanitarian text fields such as `needs`, `urgent_needs`, and `notes`, using a local LLM to generate:

- structured coding
- severity scoring
- aggregated summaries
- dashboard outputs

## Stack
- Python
- Pandas
- DuckDB
- Streamlit
- Qwen2.5 7B Instruct running locally

## Project status
Early prototype using synthetic data inspired by Lebanon-based humanitarian assessments.
